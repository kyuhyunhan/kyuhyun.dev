# Headless UI Component란?

UI가 정의되지 않고 기능적인 로직만 구현된 컴포넌트를 말한다.<br/>
컴포넌트 스타일의 커스터마이징이 자유롭고, UI의 기능을 편리하게 사용할 수 있다는 장점이 있다.

## 대표적인 React Headless-UI Library

- [React Hook Form](https://github.com/react-hook-form/react-hook-form)
