# methods

### Object.freeze()

객체(object) 선언 시 `const` 키워드를 사용하는 것만으로는 객체 속성(property)들의 불변성을 유지하지 못한다.

예를 들어,

<SyntaxHighlighter language="javascript">
  {`
    const obj = {x: 1, y: 2};

    obj = {}; // Uncaught TypeError: Assignment to constant variable.
    obj.x = 2;

`}

</SyntaxHighlighter>
와 같이, `obj` 변수가 상수로 바인딩된 객체 자체는 바꿀 수 없지만 해당 객체 값 내부의 속성값은 변경이 가능하다.

객체 내부의 속성값까지 불변하게(immutable) 설정하고 싶을 때 `Object.freeze()` 메소드를 사용하면 된다.

<SyntaxHighlighter language="javascript">
  {`
    const obj = {x: 1, y: 2};
    
    Object.freeze(obj);
    obj.x = 10;
    
    console.log(obj.x); // 1
  `}
</SyntaxHighlighter>
`strict mode` 일 경우가 아니라면 `freeze()` 메소드를 사용한 객체의 속성에 새로운
값을 재할당할 시 특별한 에러메시지가 발생하지 않는다. 그러나 해당값을 찍어보면 역시나
기존의 값이 유지되고 있음을 확인할 수 있다.

뿐만아니라 이 메소드가 사용된 객체에는 새로운 속성이 추가되지도, 기존 속성을 삭제하지도 못한다.

`Object.isFrozen()`를 이용하여 객체의 상태를 확인할 수 있다.
