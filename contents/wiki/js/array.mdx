# methods

### Array.from()

ES6에서 추가된 새로운 메소드.

유사배열(Array-like) 객체를 인자로 받아 배열 객체 인스턴스로 얕은복사를 실행한다. 여기서 유사배열이란 `length` 속성과 숫자(numeric) 키값을 인덱스처럼 가지고 있는 객체를 말한다.

객체만 인자로 받을 수도 있고, 아래와 같이 두번째 인자로 mapping 함수를 사용할 수 있다.

<SyntaxHighlighter language="javascript">
  {`
  Array.from(arrayLike, function mapFn(element, index) {/* ... */})
  Array.from(arrayLike, (element, index) => { /* ... */ } )

`}

</SyntaxHighlighter>

이 메소드가 스펙에 추가되기 전에는,

<SyntaxHighlighter language="javascript">
  {`
    Array.prototype.slice.call(arguments)
`}
</SyntaxHighlighter>
와 같이 트릭(?)을 사용했었다고 한다.{' '}

위 코드를 간단히 설명하자면, `call` 메소드를 사용해서, `slice` 메소드의 `this` 값을 `arguments`로 지정한다.
이 때 `arguments`가 유사배열이라면, `slice` 메소드를 통해 배열값으로 얕은복사가 실행되는 것이다.
